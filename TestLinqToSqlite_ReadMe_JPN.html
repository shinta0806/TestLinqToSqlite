概要

.NET Framework 4.5 C# による、LINQ to SQLite を利用したサンプルコードです。

詳細

詳細については、ブログの記事

をご覧ください。

ビルド方法

C# で SQLite を便利に使うサンプルコードのビルドには、本プロジェクトの他に、「SHINTA 共通 C# ライブラリー」内の LinqUtils.cs が必要です。

それぞれのファイル群を、例えば以下のように配置してください。本プロジェクトのファイル群よりも、SHINTA 共通 C# ライブラリーのファイル群が 2 階層上位になるように配置します。

C:\
    CsProjects\
        _Common\    ←LinqUtils.cs
        _Test\
            TestLinqToSqlite\
                Src\    ←本プロジェクトファイル群
